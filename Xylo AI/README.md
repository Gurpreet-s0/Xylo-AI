🌱 Xylo AI – Smart Farming Assistant

Xylo AI is an intelligent farming assistant that helps farmers make better decisions using AI.
Farmers can chat with the system and get guidance on crops, diseases, soil, and weather conditions.

⸻

🚀 Features
	•	💬 Chat with AI for farming advice
	•	🌾 Crop recommendations based on conditions
	•	🦠 Disease detection guidance
	•	🌦️ Weather-based suggestions
	•	⚡ Fast and responsive UI
	•	🤖 Powered by local AI models

⸻

🧠 Tech Stack

Frontend
	•	⚛️ React (Vite)
	•	🎨 Tailwind CSS

Backend
	•	🟢 Node.js
	•	🚀 Express.js

AI & Tools
